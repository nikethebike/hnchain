# HNChain Networking RFCs

Networking RFCs define peer-to-peer protocols and network behavior.

Required future RFCs include:

- P2P protocol messages
- transport abstraction
- peer identity
- handshake protocol
- capability negotiation
- gossip protocol
- block propagation
- transaction propagation
- consensus message propagation
- sync protocol
- compression profiles
- encryption profiles
- DoS protection
