# HNChain CLI RFCs

CLI RFCs define the official command-line interface for developers, operators,
wallet users, and validators.

Required future RFCs include:

- command structure
- local development environment
- wallet commands
- validator commands
- contract build and deployment
- test and sandbox execution
- configuration files
- output formats
- error codes
- API version compatibility

The CLI must call documented APIs and must not define hidden protocol behavior.
